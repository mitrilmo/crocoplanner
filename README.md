# crocoplanner
Kroko-Protokoll-App von mitrilmo 2026
Kurze Zusammenfassung des Designs:
Dieses Design fängt den Geist der "3 Krokos" ein, indem es die Logistik eines Treffens spielerisch und übersichtlich darstellt:
Identität: Euer Logo ist fest in den Header integriert.
Aktion: Der "Kroko-Check-In" Button ist das zentrale Element des Abends.
Transparenz: Die "Status"-Karte zeigt sofort, wer sein Krokodil dabei hat und wer die Konsequenzen tragen muss ("Vergessen!").
Planung: Der Countdown und die Info zum nächsten Planer sorgen dafür, dass das nächste "Schnappen" nicht in Vergessenheit gerät.
Viel Erfolg bei der Umsetzung eures Protokolls!
